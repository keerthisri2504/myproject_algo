# myproject_algo
AlgoTrack A Simple Algorand Project
Introduction
This project is a small attempt to understand how Algorand works in real-world scenarios. I am trying to build something simple that shows how fast and low-cost transactions can be done using Algorand.
The idea is to create a basic system where users can send and track small payments. It can be useful for things like splitting bills or small transactions between friends.
Uses
Fast transactions
Low fees
Beginner friendly

Just started learning and experimenting and will try to update as I go.
